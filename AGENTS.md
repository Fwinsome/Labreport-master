# Lab Report Typst SOP — 通用 AI Agent 说明

## 项目目的

将电路与电子技术实验报告的 Typst 生成流程标准化为可复用的 AI Agent 工作流。

## 适用场景

- Claude Code
- GitHub Copilot (with Custom Instructions)
- 其他支持 Skill 的 AI 助手

## 工作流阶段

| 阶段 | 阻塞条件 | 输出 |
|------|---------|------|
| 素材接收 | 须收到全部4组材料 | 确认清单 |
| 图片公式解析 | 须用户确认解析结果 | 分组规则 |
| 内容填充 | 无（可并行） | 各章节 Typst 片段 |
| 绘图拦截 | 须用户确认绘图指示 | 绘图代码 |
| 最终输出 | 须通过全部7项验证 | 完整 .typ 代码 |
